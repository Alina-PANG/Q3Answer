## Installation Guide
1. Open the MatLab and go to the `App` tab 
2. Click `Install App` 
3. Select the `ExtractMthEnd.mlappinstall` 
4. Click the newly added app icon and the `dtd_mle_2000_monthly.mat` will be generated.
